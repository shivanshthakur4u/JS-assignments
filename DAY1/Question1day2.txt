
The Console object provides access to the browser's debugging console.

Console Object Methods::

Method	Description

1.assert():	Writes an error message to the console if the assertion is false

2.clear()	:Clears the console

3.count():	Logs the number of times that this particular call to count() has been called

4.error():Outputs an error message to the console

5.group():Creates a new inline group in the console. This indents following console messages by an additional level, until console.groupEnd() is called

6.roupCollapsedd():Creates a new inline group in the console. However, the new group is created collapsed. The user will need to use the disclosure button to expand it

7.groupEnd():Exits the current inline group in the console

8.info():Outputs an informational message to the console

9.log():Outputs a message to the console

10.table():Displays tabular data as a table

11.time():Starts a timer (can track how long an operation takes)

12.timeEnd():Stops a timer that was previously started by console.time()

13.trace():Outputs a stack trace to the console

14.warn():Outputs a warning message to the console
